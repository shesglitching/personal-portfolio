# Personal Portfolio Website

A responsive personal portfolio website built using HTML, CSS, and JavaScript to showcase my education, projects, achievements, and contact information.

## Live Demo

🔗 [https://razan.wuaze.com](https://razan.wuaze.com/)

## About

This portfolio was developed as a personal website to present my background as a Computer Science student interested in embedded systems, robotics, and machine learning. The website is fully responsive and designed with a clean, modern interface.

## Features

- Responsive design for desktop, tablet, and mobile devices
- Smooth scrolling navigation
- Typing animation for the homepage title
- Scroll reveal animations
- Project showcase section
- Education and achievements section
- Contact information with GitHub, LinkedIn, and Email
- Back-to-top button

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JavaScript)
